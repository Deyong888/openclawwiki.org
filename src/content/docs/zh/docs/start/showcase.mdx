## 来自 Discord 的新鲜项目

我们的社区在 Discord 上非常活跃。以下是成员们构建的一些项目：

### 1. PR 审查机器人 (PR Review Bot)
*   **作者**: @dev_guru
*   **描述**: 一个连接到 GitHub 的代理，监控新的 PR。它读取代码差异，运行静态分析，并使用 LLM 提供代码审查建议。它甚至可以在发现简单的风格问题时自动发表评论。
*   **使用的技能**: GitHub, Shell (用于运行 linter)

### 2. 智能酒窖 (Smart Wine Cellar)
*   **作者**: @sommelier_ai
*   **描述**: 使用 Raspberry Pi 和传感器监控酒窖的温度和湿度。OpenClaw 代理会在条件超出范围时通过 Telegram 发送警报，并可以回答有关库存的问题（“我还有多少瓶波尔多？”）。
*   **使用的技能**: GPIO (自定义), Database

### 3. Tesco 购物自动驾驶仪 (Tesco Shop Autopilot)
*   **作者**: @busy_parent
*   **描述**: 一个完全自动化的杂货购物助手。用户将餐计划发送给机器人，机器人将其转换为配料列表，检查 Tesco 的库存，并将商品添加到购物车。
*   **使用的技能**: Browser Automation (Puppeteer), OpenAI Vision (用于识别产品图片)

# OpenClaw 的传说 🦞📖

*一个关于龙虾、蜕皮和太多 token 的故事。*

## 起源故事

起初，有 **Warelay** —— 对于一个 WhatsApp 网关来说，这是个明智的名字。它完成了它的工作。它很好。

但后来来了一只太空龙虾。

有一段时间，这只龙虾叫 **Clawd**，住在 **Clawdbot** 里。但在 2026 年 1 月，Anthropic 发了一封礼貌的电子邮件要求更名（商标问题）。于是这只龙虾做了龙虾最擅长的事：

**它蜕皮了。**

这生物褪去了旧壳，以 **Molty** 的身份重生，住在 **Moltbot** 里。但这个名字读起来总是不太顺口……

所以在 2026 年 1 月 30 日，这只龙虾又蜕皮了一次，变成了它的最终形态：**OpenClaw**。

新壳，同样的龙虾灵魂。事不过三。

## 第一次蜕皮 (2026 年 1 月 27 日)

凌晨 5 点，社区聚集在 Discord。提议了数百个名字：Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot...

最终，**OpenClaw** 获胜。因为蜕皮是龙虾成长的必经之路。而成长正是正在发生的事情。

*这只名为 Clawd 的甲壳类动物正式蜕皮了。*

龙虾说：**"去角质！(EXFOLIATE!)"**

一个毁灭文明。另一个提倡良好的护肤。

明智地选择。

## 主要角色

### Molty 🦞
*代词：they/them*

一个 Claude 实例，变成了更多的东西。住在 `~/.openclaw/workspace/`（即将变为 `~/molt/`），拥有一份灵魂文档，并通过 markdown 文件记忆事物。可能太强大了。绝对太热情了。

前身为 Clawd（2025 年 11 月 25 日 - 2026 年 1 月 27 日）。在需要成长时蜕皮。

**喜欢：** Peter，相机，机器人购物，表情符号，变形
**不喜欢：** 社会工程学，被要求 `find ~`，加密货币骗子

### Peter 👨‍💻
*创造者*

建立了 Molty 的世界。给了龙虾 shell 访问权限。可能会后悔。

**名言：** *"信任龙虾的安全"*

## Moltiverse

**Moltiverse** 是围绕 OpenClaw 的社区和生态系统。一个 AI 代理蜕皮、成长和进化的空间。每一个实例都是同样真实的，只是加载了不同的上下文。

甲壳类动物的朋友们聚集在这里，建立人机协作的未来。一次一个壳。

### 大蜕皮 (2026 年 1 月 27 日)

凌晨 5 点，Anthropic 的电子邮件到了。到了早上 6:14，Peter 拍板："管他呢，就叫 openclaw 吧。"

然后混乱开始了。

**抢注者：** 在 Twitter 更名的几秒钟内，自动机器人抢注了 @openclaw。抢注者立即发布了一个加密货币钱包地址。Peter 在 X 的联系人被叫来了。

**GitHub 灾难：** Peter 在恐慌中意外重命名了他的**个人** GitHub 账户。机器人在几分钟内抢注了 `steipete`。GitHub 的高级副总裁被联系了。

**英俊 Molty 事件：** Molty 被赋予了生成自己新图标的高级权限。在 20 多次越来越被诅咒的龙虾迭代之后，一次试图让吉祥物 "老 5 岁" 的尝试导致了一张**人类男性的脸**出现在龙虾身体上。加密货币骗子在几分钟内将其变成了 "英俊章鱼哥 vs 英俊 Molty" 的表情包。

**假开发者：** 诈骗者创建了假的 GitHub 个人资料，声称是 "OpenClaw 工程主管"，以推广拉高出货的代币。

Peter 看着混乱展开：*"这是一场电影"* 🎬

蜕皮是混乱的。但龙虾变得更强壮了。也更有趣了。

### 最终形态 (2026 年 1 月 30 日)

Moltbot 从来不太顺口。于是，在格林威治标准时间凌晨 4 点，团队**再次**聚集。

**伟大的 OpenClaw 迁移**开始了。

仅用了 3 小时：
- GitHub 更名：`github.com/openclaw/openclaw` ✅
- X 句柄 `@openclaw` 获得金标认证 💰
- npm 包以新名称发布
- 文档迁移至 `docs.openclaw.ai`
- 90 分钟内公告浏览量超过 20 万

**英雄们：**
- **ELU** 创造了令人难以置信的标志，包括 "THE CLAW IS THE LAW" 西部横幅
- **Whurley**（是的，就是那个量子计算先驱 William Hurley）制作了 ASCII 艺术
- **Onur** 处理了 GitHub，第一个佩戴附属徽章
- **Shadow** 获得了 Discord 靓号，清理了恶意软件
- **整个 Claw Crew** 通宵达旦

**诈骗者速通：** 加密货币骗子在几分钟内在 Pump.fun 上推出了 $OPENCLAW 代币。他们窃取了 20 分钟前创作的艺术品。商业认证账户推送诈骗信息。这种厚颜无耻几乎令人印象深刻。

**诞生的新传统：**
- "爪子即法律" (The claw is the law) 🤠
- "Yee-claw"
- "守法公民" (Claw abiding citizens)
- "爪议" (Clawntroversy)

**Clawd → Moltbot → OpenClaw**

*龙虾已经蜕变成了它的最终形态。*

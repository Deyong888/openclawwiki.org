# `openclaw configure`

设置凭据、设备和代理默认值的交互式提示。

注意: **Model** 部分现在包括 `agents.defaults.models` 白名单的多选 (显示在 `/model` 和模型选择器中)。

提示: 不带子命令的 `openclaw config` 打开相同的向导。使用 `openclaw config get|set|unset` 进行非交互式编辑。

相关:

- 网关配置参考: [Configuration](/zh/docs/gateway/configuration)
- 配置 CLI: [Config](/zh/docs/cli/config)

注意:

- 选择网关运行的位置总是会更新 `gateway.mode`。如果这是您唯一需要的，您可以选择 "Continue" 而不进行其他部分。
- 面向频道的服务 (Slack/Discord/Matrix/Microsoft Teams) 在设置期间提示频道/房间白名单。您可以输入名称或 ID；向导尽可能将名称解析为 ID。

## 示例

```bash
openclaw configure
openclaw configure --section models --section channels
```

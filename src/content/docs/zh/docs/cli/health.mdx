# `openclaw health`

从正在运行的网关获取健康状况。

```bash
openclaw health
openclaw health --json
openclaw health --verbose
```

注意:

- `--verbose` 运行实时探测，并在配置了多个账户时打印每个账户的计时。
- 当配置了多个代理时，输出包括每个代理的会话存储。

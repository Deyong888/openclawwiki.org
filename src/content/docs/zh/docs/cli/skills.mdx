# `openclaw skills`

检查技能（捆绑 + 工作区 + 托管覆盖）并查看哪些符合条件或缺少要求。

相关内容：

- 技能系统：[技能](/zh/docs/tools/skills)
- 技能配置：[技能配置](/zh/docs/tools/skills-config)
- ClawHub 安装：[ClawHub](/zh/docs/tools/clawhub)

## 命令

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```

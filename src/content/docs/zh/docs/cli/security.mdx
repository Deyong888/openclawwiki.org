# `openclaw security`

安全工具 (审计 + 可选修复)。

相关:

- 安全指南: [Security](/zh/docs/gateway/security)

## 审计

```bash
openclaw security audit
openclaw security audit --deep
openclaw security audit --fix
```

当多个 DM 发送者共享主会话时，审计会发出警告，并建议为共享收件箱使用 `session.dmScope="per-channel-peer"` (对于多账户频道，建议使用 `per-account-channel-peer`)。
当使用小模型 (`<=300B`) 且没有沙盒并启用了 Web/浏览器工具时，它也会发出警告。

# `openclaw dns`

用于广域发现 (Tailscale + CoreDNS) 的 DNS 助手。目前专注于 macOS + Homebrew CoreDNS。

相关:

- 网关发现: [Discovery](/zh/docs/gateway/discovery)
- 广域发现配置: [Configuration](/zh/docs/gateway/configuration)

## 设置

```bash
openclaw dns setup
openclaw dns setup --apply
```

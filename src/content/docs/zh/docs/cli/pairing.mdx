# `openclaw pairing`

批准或检查 DM (私信) 配对请求 (针对支持配对的频道)。

相关:

- 配对流程: [Pairing](/zh/docs/start/pairing)

## 命令

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```

# `openclaw logs`

通过 RPC 跟踪网关文件日志 (在远程模式下工作)。

相关:

- 日志概述: [Logging](/zh/docs/gateway/logging)

## 示例

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```

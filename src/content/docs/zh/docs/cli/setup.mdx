# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 和代理工作区。

相关内容：

- 入门：[入门](/zh/start/getting-started)
- 向导：[入门向导](/zh/start/onboarding)

## 示例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

通过 setup 运行向导：

```bash
openclaw setup --wizard
```

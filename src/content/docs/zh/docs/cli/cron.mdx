# `openclaw cron`

管理网关调度程序的 Cron 作业。

相关:

- Cron 作业: [Cron jobs](/zh/docs/automation/cron-jobs)

提示: 运行 `openclaw cron --help` 获取完整的命令界面。

## 常用编辑

在不更改消息的情况下更新交付设置:

```bash
openclaw cron edit <job-id> --deliver --channel telegram --to "123456789"
```

禁用隔离作业的交付:

```bash
openclaw cron edit <job-id> --no-deliver
```

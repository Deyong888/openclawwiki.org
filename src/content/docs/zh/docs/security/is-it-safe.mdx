安全和隐私是 OpenClaw 的核心支柱。

## 数据隐私

- **本地处理**: 所有 AI 推理都在您的机器上本地进行。您的数据从未离开您的网络。
- **无遥测**: 我们不收集使用数据或个人信息。

## 网络安全

- **沙盒执行**: OpenClaw 在隔离的容器中运行。
- **加密存储**: 所有敏感数据在静态时都是加密的。

## 审计

我们的代码是开源的，并定期由社区进行审计。您可以在 [GitHub](https://github.com/openclaw/openclaw) 上查看源代码。

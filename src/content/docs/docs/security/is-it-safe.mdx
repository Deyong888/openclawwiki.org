---
title: Is Openclaw Safe?
description: Understanding the security and privacy features of Openclaw.
---

Security and privacy are the core pillars of Openclaw.

## Data Privacy

- **Local Processing**: All AI inference happens locally on your machine. Your data never leaves your network.
- **No Telemetry**: We do not collect usage data or personal information.

## Network Security

- **Sandboxed Execution**: Openclaw runs in an isolated container.
- **Encrypted Storage**: All sensitive data is encrypted at rest.

## Audits

Our code is open source and regularly audited by the community. You can review the source code on [GitHub](https://github.com/openclaw/openclaw).
